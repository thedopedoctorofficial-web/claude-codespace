# Claude Code Workspace

Open deze repo in GitHub Codespaces om Claude Code te gebruiken vanaf elk apparaat.

## Gebruik
1. Klik op **Code** → **Codespaces** → **Create codespace on main**
2. Wacht tot de container klaar is (~2 min)
3. Open de Terminal en typ `claude`

## Gratis tier
- 60 uur/maand op 2-core machine
- 15 GB opslag/maand
